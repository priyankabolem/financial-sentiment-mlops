# 🚀 Deploy Your Live Demo NOW (5 Minutes)

## Quick Deploy to Hugging Face Spaces

### Step 1: Go to Hugging Face (30 seconds)
1. Open: https://huggingface.co
2. Click "Sign Up" (or "Log In" if you have account)
3. Complete signup with email/GitHub

### Step 2: Create Space (1 minute)
1. Click your profile picture (top right)
2. Click "New Space"
3. Fill form:
   ```
   Owner: [your username]
   Space name: financial-sentiment-analysis
   License: MIT
   Select SDK: Streamlit
   Space hardware: CPU basic - Free
   Visibility: Public
   ```
4. Click **"Create Space"**

### Step 3: Upload Files (2 minutes)

**Upload these 3 files:**

1. **Click "Files" tab → "Add file" → "Upload files"**

2. **Upload File 1:** `app.py`
   - Drag and drop: `app.py`

3. **Upload File 2:** Rename `requirements-demo.txt` to `requirements.txt`
   - Go back to main repo
   - Copy content from `requirements-demo.txt`
   - In HF Space: "Add file" → "Create a new file"
   - Name it: `requirements.txt`
   - Paste content
   - Commit

4. **Upload File 3:** Create `README.md`
   - "Add file" → "Create a new file"
   - Name it: `README.md`
   - Copy content from `README_SPACES.md`
   - Paste and commit

### Step 4: Wait for Build (2 minutes)
- Space will auto-build
- Watch logs in "Logs" tab
- When you see "App running" → You're LIVE! 🎉

### Step 5: Test Your Demo (1 minute)
1. Click "App" tab
2. Your demo is live!
3. Try examples
4. Share the URL!

## 🆘 Troubleshooting

**Problem:** Build fails
- **Solution:** Check requirements.txt format (no extra spaces)

**Problem:** App doesn't load
- **Solution:** Wait 3-4 minutes for model download on first run

**Problem:** Out of memory
- **Solution:** Should work on free tier, if not contact HF support

**Need help?** Open issue on GitHub repo!
